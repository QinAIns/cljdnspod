# cljdnspod

DNSPOD的Clojure版本，每隔10分钟动态修改域名的ip。

## 用法
1. 去这里 [https://www.dnspod.cn/docs/info.html#api-token](https://www.dnspod.cn/docs/info.html#api-token) 查看获取token的文档

2. 修改src/cljdnspod/config.clj里边的相关配置

3. 之后在命令行下执行：lein run
